Visualizing historical rainfall - Part 2

17 Apr, 2024

In 2023, I made a visualization of the historical rainfall over my farm to help me improve my farming practice. This was made using the the Observable tool.

Now in 2024, Observable folks released the Observable Framework, which is a Javascript framework that can be used to create standalone visualizations and built into a static website. This sounded exciting and I decided to take another stab at rainfall visualization using this framework.

You can see it embedded below.

You can also access the standalone website. The source code is available on Github.
